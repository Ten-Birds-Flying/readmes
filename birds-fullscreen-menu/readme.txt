== Description ==

Birds Fullscreen Menu provides an additional Fullscreen Menu for your website.

This plugin is Localization/Internationalization ready and follows WordPress i18n standards.


== Installation ==

1. Unzip plugin files and upload them under your '/wp-content/plugins/' directory.
2. Resulted names will be: '/wp-content/plugins/birds-fullscreen-menu/'
3. Activate plugin at "Plugins" administration page.


== Frequently Asked Questions ==

= Where are the settings of the plugin? =

Just go to Appearance > Customize
