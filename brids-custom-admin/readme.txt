== Description ==

Birds Custom Admin allows you to easily customize WordPress admin panel, admin menu, admin bar etc.

This plugin is Localization/Internationalization ready and follows WordPress i18n standards.


== Installation ==

1. Unzip plugin files and upload them under your '/wp-content/plugins/' directory.
2. Resulted names will be: '/wp-content/plugins/birds-custom-admin/'
3. Activate plugin at "Plugins" administration page.


== Frequently Asked Questions ==

= Can I revert to original WordPress Admin? =

Yes, just switch off the plugin in the Activation Section.
